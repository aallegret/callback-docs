mediaError
==========

Une fonction de callback définie par l'utilisateur et qui est appelée lorsqu'une erreur survient lors d'un appel d'une des méthodes de `Media`.

    function(error) {
        // Gérer l'erreur
    }

Paramètres
----------

- __error:__ L'erreur retournée par l'appareil. (`MediaError`)
